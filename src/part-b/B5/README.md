# B5: パスワードのリアルタイム検証

パスワード入力欄1つだけを持つフォームを作成し、入力中にリアルタイムでバリデーションを行いなさい。

- パスワード欄は1つのみとすること
- 入力中に、各必須条件を満たしているかをリアルタイムで表示すること
- 必須条件は以下の通りとすること
- 8文字以上16文字以下であること
- 英大文字を1文字以上含むこと
- 英小文字を1文字以上含むこと
- 数字を1文字以上含むこと
- 記号を1文字以上含むこと
- スペースを含まないこと
- すべての条件を満たした場合のみ「利用可能」などの成功状態を表示すること
- 条件を満たしていない場合は、未達成の条件が分かること
- 送信ボタンは、すべての条件を満たしたときのみ有効化すること
